📖 Guide d'utilisation

🚀 Comment utiliser Bibile ?

1

Ouvrez votre fichier PDF

Ouvrez le fichier PDF que vous recevez chaque jour par email

2

Sélectionnez tout le contenu

Appuyez sur Ctrl+A pour tout sélectionner

3

Copiez le texte

Appuyez sur Ctrl+C pour copier

4

Collez dans Bibile

Cliquez dans le grand champ de texte et appuyez sur Ctrl+V

5

Générez le fichier Excel

Cliquez sur le bouton "Générer le fichier Excel"

6

Téléchargez et utilisez

Le fichier Excel se télécharge automatiquement dans votre dossier Téléchargements

📊 Visualisation des données (page "Données")

Après avoir généré un fichier Excel, vous pouvez consulter et analyser les données directement dans votre navigateur :

1

Accédez à la page Données

Cliquez sur "Données" dans le menu de navigation

2

Sélectionnez un fichier

Choisissez un fichier Excel dans la liste déroulante

3

Utilisez les filtres

Filtrez par livraison, ville, référence ou recherche globale

Fonctionnalités de la page Données :

  • Statistiques en temps réel - Totaux par type de livraison (BREVET, TRANSIT, CHEVROLET)
  • Codes couleurs - Chaque livraison a une couleur distincte (bleu, orange, vert)
  • Filtrage instantané - Les résultats s'affichent immédiatement sans rechargement
  • Recherche globale - Cherchez dans toutes les colonnes simultanément
  • Statistiques dynamiques - Les totaux se mettent à jour automatiquement avec les filtres

📋 Les colonnes du tableau :

  • N° Enl. - Numéro de l'enlèvement
  • Référence - Référence interne
  • Société - Nom de la société
  • Ville - Ville de l'enlèvement
  • Nb Pal. - Nombre de palettes (PART=0, HALF=1)
  • Type Pal. - VMF, EURO, PART PALLET, HALF PALLET
  • Poids (kg) - Poids total en kilogrammes
  • Nb Colis - Nombre de colis
  • Livraison - BREVET, TRANSIT, CHEVROLET
  • Tél. - Numéro de téléphone

🎨 Codes couleurs des livraisons

Dans la page Données, chaque type de livraison a une couleur distinctive :

  • 🟦 BREVET - Bordure bleue
  • 🟧 TRANSIT - Bordure orange
  • 🟩 CHEVROLET - Bordure verte

Ces couleurs facilitent l'identification rapide des types de livraison dans le tableau.

🔍 Deux façons de filtrer vos données

Option 1 : Filtres dans la page Données (recommandé)

  • Allez sur la page Données
  • Sélectionnez votre fichier
  • Utilisez les filtres instantanés (livraison, ville, référence, etc.)
  • Les statistiques se mettent à jour automatiquement

Option 2 : Filtres dans Excel

Le fichier Excel téléchargé contient aussi des filtres automatiques :

  1. Ouvrez le fichier Excel téléchargé
  2. Cliquez sur les petites flèches dans les en-têtes de colonnes
  3. Décochez/Cochez les valeurs que vous voulez afficher
  4. La ligne TOTAL en bas se met à jour automatiquement

⚠️ En cas de problème

❌ "Aucun enlèvement trouvé"

Solution : Vérifiez que vous avez bien copié TOUT le contenu du PDF. Le texte doit contenir des lignes comme "Enlèvement 1", "Enlèvement 2", etc.

❌ "Erreur lors de la génération"

Solution : Rechargez la page et réessayez. Si le problème persiste, contactez le support informatique.

❌ Le fichier ne se télécharge pas

Solution : Vérifiez les paramètres de votre navigateur. Assurez-vous que les téléchargements automatiques sont autorisés.

❌ La page Données ne charge pas

Solution : Vérifiez qu'un fichier Excel a bien été généré. Allez sur la page Historique pour voir les fichiers disponibles. Si aucun fichier n'existe, générez-en un depuis la page d'accueil.

❌ Les statistiques sont incorrectes

Solution : Cliquez sur "Réinitialiser" pour effacer tous les filtres. Vérifiez que le fichier source est correct. Les statistiques comptent le nombre de palettes (PART=0, HALF=1).

💡 Astuces

  • Consultez la page Données pour analyser vos fichiers sans ouvrir Excel
  • Consultez l'Historique pour télécharger vos anciens fichiers Excel
  • Le bouton "Effacer" permet de vider rapidement le champ de texte
  • Utilisez Ctrl+Enter comme raccourci pour générer le fichier
  • Les filtres de la page Données sont instantanés (pas besoin de recharger)
  • Les statistiques se mettent à jour automatiquement quand vous filtrez les données
  • Chaque génération est enregistrée dans les logs pour traçabilité
