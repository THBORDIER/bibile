DEMARRAGE RAPIDE :
------------------

1. Allez dans le dossier "bibile/"

2. Double-cliquez sur "INSTALLER.bat"
   -> Installe les dependances et cree le raccourci sur le bureau

3. Double-cliquez sur l'icone "Bibile" qui apparait sur votre bureau
   -> Le navigateur s'ouvre automatiquement

4. Collez le texte du PDF et cliquez sur "Generer Excel"

C'est tout !
